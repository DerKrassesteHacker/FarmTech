
# FarmTech

Software für den ESP8266 für das FarmTech-Projekt. Der ESP8266 liest Sensoren-Daten und schickt sie an den Raspberry-Pi-Zero-Redis-Server, wo sie gespeichert werden und für die Handy-App abrufbar sind.
